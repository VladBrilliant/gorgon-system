## [0.0.1] — 2025-11-27

### Added

- Initial project skeleton
- `pyproject.toml` for editable installation
- `RhopaliumCPU` and `RhopaliumMemory` sensors
- `CrabGuardian` monitoring unit
- `Octopus` core hub
- `GorgonBellCLI` CLI layer
- Example scripts:
  - `demo_cpu_mem.py`
  - `demo_octopus_cpu_mem.py`
  - `demo_bell_cli.py`
- Basic documentation:
  - `README.md`
  - `docs/MANUAL.md`
  - `docs/ARCHITECTURE.md`
  - `docs/ROADMAP.md`
  - `docs/CHANGELOG.md`
